# 背景画像フォルダ

このフォルダに背景画像を入れると、子機（回答書き画面）の背景として選択できます。

## 対応フォーマット
- PNG（推奨）
- JPG / JPEG
- WEBP
- GIF

## TIFFについて
ブラウザはTIFF形式を直接表示できません。
TIFFファイルはあらかじめ **PNG に変換** してからこのフォルダに入れてください。

## 使い方
1. 画像ファイルをこのフォルダに配置する
2. サーバーを起動する（すでに起動中なら再起動不要）
3. 子機画面のツールバー「🖼️ 背景」ボタンから選択する
4. 「なし」を選択すると背景をクリアできる

## 推奨サイズ
タブレットの解像度に合わせた横長画像（例: 1920×1080、1280×720）が綺麗に表示されます。
